# [예산 작업] 신규 화면 및 기능 반영
취합 및 결재가 완료된 예산 목록을 기반으로 예산 작업을 하는 화면과 기능을 구현해줘

 - 화면명 : 예산 작업
 - 주요 기능
  1. [편성률 입력] 각 비목별(공통코드의 코드값구분이 DUP_IOE 모든 비목 유형)로 [편성률]을 입력
   1-1. 편성률은 0~100 사이의 값
   1-2. 편성률이 0이면 해당 비목은 예산에서 제외
   1-3. 편성률이 100이면 해당 비목은 예산에 포함
   1-4. 편성률이 0~100 사이이면 해당 비목은 예산에 포함
  2. [편성률 적용] 각 사업의 비목별 금액에 적용
   2-1. 데이터 예시 : 비목코드값 (237-0700) - 편성비목 공통코드 ID (DUP-IOE-237)
   2-2. 편성비목 공통코드 ID에서 'DUP-IOE-'를 제외한 값 237로 시작하는 비목코드를 확인하는 방식으로 적용
  3. [편성률 입력 후 저장 버튼 클릭 시] 비목별 예산 현황을 테이블로 보여줌
   3-1. 비목 / 요청금액 / 편성금액 / 편성률
