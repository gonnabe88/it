
## 전산업무비 목록 화면 개선

### 개요
/info/cost/ 화면에서 [금융정보단말기 상세목록] 까지 작성 할 수 있도록 하단에 서브 테이블을 구성하고 이에 따른 불필요한 기능 제거

### 세부사항
1. /info/cost/ 화면 [전산업무비 목록]의 금융정보단말기 하단에 [금융정보단말기 상세목록] 서브 테이블을 추가 
2. [금융정보단말기 상세목록] 테이블은 상단 [전산업무비 목록] 기본 테이블에서 선택한 컬럼에 대한 금융정보단말기 목록을 보여줌
3. [금융정보단말기 상세목록] 테이블에서 데이터 추가, 수정, 삭제 가능
4. /info/cost/terminal/form 화면은 더이상 필요 없으므로 삭제
5. [전산업무비 목록] 기본 테이블에서는 Cell 단위 '인라인 편집' 기능을 제거하고 대신 Row를 선택할 수 있도록 수정 (데이터 수정은 '수정' 아이콘을 통해 Row 단위로 수정)
