3. /budget/status (예산현황)에서 전산업무비는 [편성요청] / [조정(편성)]을 [2025년] / [2026년]으로 변경하고 편성 결과 금액이 보이도록 수정

4. /info/cost 전산업무비 목록 화면에서 유형 컬럼을 없애고 금융정보단말기 여부 컬럼(체크박스) 추가
  - 금융정보단말기 여부 컬럼은 체크박스로 표시
  - 금융정보단말기 여부 체크 시 /info/cost/terminal/form 상세화면으로 이동하여 작성이 가능하도록 개선 (현재는 /info/cost/terminal/form 신규 작성은 불가)
  - 금융정보단말기 /info/cost/terminal/form 상세화면에서 작성 완료 시 해당 탭을 닫고 전산업무비 목록 화면으로 돌아오도록 수정

5. /info/cost, /info/cost/terminal/form 화면에서 테이블 인라인 수정 시 항상 편집 가능한 모드로 보이게 하지 않고, 해당 컬럼 값을 클릭하면 해당 컬럼만 편집 가능한 모드로 바뀌도록 수정
  - 편집 가능한 모드에서 엔터키를 누르면 저장되고, esc 키를 누르면 취소되도록 수정
  - 저장 또는 취소 시 편집 가능한 모드가 해제되도록 수정

6. 전산예산 신청 가능기간 설정
 - 공통코드에서 설정한 기간 외에는 /budget 화면 접속 시 안내 팝업(다이얼로그) 표시
 - 공통코드에서 설정한 기간 외에는 /budget 화면에서 [정보화사업], [전산업무비], [경상사업] 버튼 비활성화
 - 공통코드에서 설정한 기간 외에는 [정보화사업], [전산업무비], [경상사업] 작성(form) 화면을 직접 접속하는 경우에도 접근 불가하도록 수정
