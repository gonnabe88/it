# 하드코딩 제거 및 코드 적용
아래의 항목에 대해 코드 기반으로 목록을 고를 수 있도록 적용해줘

## /info/cost/form
1. 비목코드
 - 코드유형(CTT_TP) : IOE_LEAFE, IOE_XPN, IOE_SEVS, IOE_IDR
 - 화면표기 : C_NM (코드명)
 - 설정값 : C_ID (코드ID)

2. 구분
 - 코드유형(CTT_TP) : CTT_TP
 - 화면표기 : C_NM (코드명)
 - 설정값 : C_ID (코드ID)

 3. 지급주기
 - 코드유형(CTT_TP) : DFR_CLE
 - 화면표기 : C_NM (코드명)
 - 설정값 : C_ID (코드ID)
