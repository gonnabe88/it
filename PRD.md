# 사업 가이드 Editor 고도화

사업 가이드 (/guide) 화면의 Editor를 고도화

## 정렬 기능 정상화
1. 현재 정렬 후 저장 시 정렬이 유지되지 않는데, 저장 후에도 유지되도록 개선

## 체크박스 표기 오류 개선
1. 현재 체크박스 추가 후 저장 시 불릿 기호로 표시되는데, 체크박스로 표시되도록 개선

## 코드 블록 개선
1. 현재 코드 블록 내부 언어 설정이 불가한데, 언어 설정 시 해당 언어에 맞게 하이라이팅 되도록 개선

## 폰트 크기 조절 기능 추가
1. 폰트 크기를 조절할 수 있는 기능 추가
2. 폰트 크기는 10px ~ 24px 까지 조절 가능
3. 폰트 크기는 2px 단위로 조절 가능

## 첨부파일 기능 추가
1. /api/files/ API를 활용하여 첨부파일 업로드 (삭제, 다운로드 기능 포함)
2. Tiptap Editor에서 ![]() 형태로 첨부파일을 삽입할 수 있도록 기능 추가
3. ![ 까지 입력 시 첨부파일 목록을 보여주고, 선택 시 ![]() 형태로 삽입
4. 첨부파일은 해당 라인 오른쪽 끝에 표시(우측 상단 & 좌측 하단만 라운드 진한 indigo색 하얀색 글자 박스 모양)하고, 클릭 시 다운로드

## Editor Table 기능 추가
1. 테이블 정렬 기능 추가
2. 테이블 테두리 Style 설정 기능 추가
3. 테이블 셀배경 색상 팔레트 변경 (기존 RGB 방식 > 변경 팔레트 방식)
4. 테이블 반응형/고정형 전환 기능 추가
5. 테이블 셀 높이 조절 기능 추가

## mathlive(https://github.com/arnog/mathlive) LaTeX 수식 기능 추가
1. mathlive를 활용하여 LaTeX 수식을 입력할 수 있는 기능 추가
2. LaTeX 수식은 저장 후에도 유지되도록 구현
